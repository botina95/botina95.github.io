🔍 OCR & Extracción de Datos

Procesa el documento una vez.
Extrae datos infinitas veces.

MAPOCR aplica OCR al documento, almacena el resultado indexado y permite aplicar cualquier plantilla de extracción cuantas veces necesites sin volver a procesar el archivo. Disponible como API REST y como cliente CLI.

  • ✓Elige el motor: Tesseract local (sin costos extras, 100% offline) o GPT-4o IA (mayor precisión en docs complejos, con costo adicional)
  • ✓API REST + CLI: integra MAPOCR en cualquier sistema o pipeline desde terminal
  • ✓Cola asíncrona con worker en background para volúmenes altos sin bloquear el sistema
  • ✓Plantillas reutilizables con reglas por campo; también extracción ad-hoc sin plantilla guardada
  • ✓Perfiles de preprocesamiento: corrección de imagen, rotación automática, binarización configurables
  • ✓Dashboard de métricas: jobs por día, volúmenes, documentos más grandes, estadísticas globales
  • ✓Referencias externas para enlazar resultados con tu GED sin modificar sus identificadores internos
  • ✓Formatos: PDF, PNG, JPG, TIFF · BD cifrada SQLCipher · Swagger incluido

🗃️ Gestión Documental

Digitaliza, indexa y exporta.
Listo para tu sistema GED.

INDEXA convierte documentos físicos en registros digitales estructurados con validación de datos, reglas por campo y scripts C# personalizados. Resultado: índice TXT + imagen listo para importar en OnBase u otro sistema documental.

  • ✓Scripts C# (Roslyn) a nivel documento y campo: autollenado, validaciones cruzadas, lógica personalizada sin límites
  • ✓Listas en cascada multi-nivel filtradas automáticamente según la selección anterior
  • ✓Tipos de documento configurables con campos, reglas y scripts completamente independientes
  • ✓Drivers TWAIN y WIA — compatible con la mayoría de escáneres del mercado
  • ✓Control de calidad (QA): calificación de lotes, observaciones y dashboard de productividad
  • ✓Pools de licencia Digit e Index independientes, asignados por máquina (HWID)
  • ✓Grupos de exportación configurables para el formato exacto que espera cada sistema destino
  • ✓Bloqueo concurrente de lotes · Documentos multipágina · Roles y permisos granulares

⚙️ Captura & Optimización

Lee cualquier pantalla.
Consulta. Decide más rápido.

CAPTOR captura datos de cualquier aplicación de escritorio o web, los consulta contra tus fuentes de información y presenta el resultado al operario en segundos. Menos búsquedas manuales, decisiones más ágiles.

  • ✓Captura visual universal: selecciona con clic cualquier campo de app desktop o web — sin programar selectores
  • ✓Consulta bases de datos, sistemas externos vía API REST o Script C# personalizado — las posibilidades son infinitas
  • ✓7 proveedores de datos: SQL Server, MySQL, SQLite, REST API, Excel, Archivo de texto, Script C#
  • ✓Conecta APIs externas como proveedor de datos: Bearer, Basic Auth, token dinámico, headers y body JSON/Form
  • ✓8 tipos de respuesta: mensaje, tabla, campos capturados, script, combinado, abrir URL, carpeta o archivo
  • ✓Hotkey global: el operario activa el perfil con una tecla sin salir de su aplicación actual
  • ✓Encadenamiento de acciones: el resultado de una consulta alimenta la siguiente en el mismo flujo
  • ✓Log de ejecuciones · Roles y grupos · Sesiones en tiempo real · Vive en la bandeja del sistema
